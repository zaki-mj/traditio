# Traditional Gems â€“ Discover Algeria's cultural places

**Traditional Gems** is a Flutter app for discovering traditional and cultural places across Algeria such as hotels, restaurants, attractions and local stores. The app was developed as a client request for PhD research and supports bilingual/localized content for real-world field studies.

---

## Highlights

- **Discover Locations**: Browse hotels, restaurants, attractions and local places by city or region
- **Detailed Listings**: View photos, descriptions, and helpful metadata for each place
- **Admin Dashboard**: Add or update POI entries through a simple admin panel
- **Localization**: Arabic, French, and English support
- Lightweight and fast, optimized for mobile usage in regions with varying connectivity
- **Admin Auth**: Secure login for administrators (email/password or integrated backend)
- Uses structured local assets for offline city data and media
- Firestore-backed POI model for real-time data
- Local favorites stored with SharedPreferences
- Category-based UI fallbacks and responsive design

## Getting Started

### Prerequisites

- Flutter SDK installed
- Dart SDK
- Connected device or emulator

### Run the app (quick)

```bash
git clone https://github.com/zaki-mj/traditio.git
cd traditio
flutter pub get
flutter run
```
